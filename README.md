# vk-copy-inner-block

## これは何？

このブロックの中にあるブロックのコードをコピーするためのブロックです。

コードは下記にあります。
https://github.com/vektor-inc/vk-copy-inner-block

パッケージのインストール
```
$ npm install
$ composer install
```

js,sassの監視を開始します。
```
$ npm run watch
```
